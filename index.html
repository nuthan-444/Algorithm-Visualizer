◈ Dynamic Programming DP Concepts & Visualizations

🔁 Recursion

Solves the problem by dividing into subproblems repeatedly. May recompute the same subproblem multiple times.

Exponential: O(2ⁿ)

📝 Memoization (Top-Down)

Recursion + cache. Store results of computed subproblems. Only computes each unique subproblem once.

Polynomial: O(n)

📊 Tabulation (Bottom-Up)

Fill a DP table iteratively from base cases. No recursion stack overhead. Preferred in practice.

Optimal: O(n) time & O(n) space
